AdicionarTemplate Adiciona uma digital a um usuário que já tenha sido adicionado à lista em memória (com a função AdicionarUsuario(Int64, String, UInt32, String, String, UInt64, Int32)). Os dados serão gravados em arquivo quando a função FinalizarGravacao(String, String) for chamada.
AdicionarTemplateStr Adiciona uma digital a um usuário que já tenha sido adicionado à lista em memória (com a função AdicionarUsuario(Int64, String, UInt32, String, String, UInt64, Int32)). Os dados serão gravados em arquivo quando a função FinalizarGravacao(String, String) for chamada.
AdicionarUsuario Adiciona um usuário à lista em memória de usuários que serão gravados em um arquivo quando a gravação for finalizada (função FinalizarGravacao(String, String)).
CarregarTemplatesUsuario Busca na memória as digitais do usuário especificado pelo número PIS, e essas digitais poderão ser lidas a seguir com qualquer uma das funções LerTemplate(Byte) ou LerTemplateStr(String).
CarregarUsuarios Le o conteudo de arquivos de usuários e armazena na memória. Para buscar os registros lidos, utilize as funções LerUsuario(Int64, String, UInt32, String, String, UInt64, Int32, Int32), CarregarTemplatesUsuario(Int64, Int32), LerTemplate(Byte) e LerTemplateStr(String).
FinalizarGravacao Grava no arquivo especificado os usuários que foram incluídos anteriormente com as funções AdicionarUsuario(Int64, String, UInt32, String, String, UInt64, Int32), AdicionarTemplate(Int64, Byte) e AdicionarTemplateStr(Int64, String)
GetLastError Retorna a descrição do último erro que tenha causado uma função a retornar false.
IniciaGravacao Reseta o estado interno da DLL para começar a gravar dados de usuários em Pen Drive.
LerTemplate Lê uma a uma as digitais do funcionário especificado anteriormente ao chamar a função CarregarTemplatesUsuario(Int64, Int32)
LerTemplateStr Lê uma a uma as digitais do funcionário especificado anteriormente ao chamar a função CarregarTemplatesUsuario(Int64, Int32)
LerUsuario Lê um a um os dados dos funcionários lidos na memória com a função CarregarUsuarios(String, String, Int32, Boolean).
